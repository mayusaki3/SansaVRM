# SansaVRM とは

従来のVRMは主にヒューマノイドモデルを対象として設計されており、非人型モデル（ロボット、昆虫、クモ、抽象オブジェクト等）への対応は限定的である。
また、モデル構造、装着、権利管理、収益分配などが分断されており、プラットフォームとしての拡張性に課題がある。

そこで、SansaVRM では、以下を実現するための統合モデル基盤を構築する。

- あらゆる構造を持つモデルの表現（ヒューマノイド非依存）
- モジュール分割による柔軟な構成
- モデル・装着・権利・収益の統合管理
- Runtime における動的構成
- ワールドオブジェクトとの統合
- 経済圏の構築
- 表情、材質変化、テクスチャ変化を含む拡張表現の統合管理
- モジュールやアクセサリーの状態切替・着脱切替の統合管理
